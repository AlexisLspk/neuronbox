# Licensing

**Dual licensing model (AGPL v3 + commercial license)**

This project is released under a dual licensing model designed to balance open innovation with sustainable development.

---

## Open source license — AGPL v3

The core of this project is licensed under the **GNU Affero General Public License v3** (AGPL v3). The complete legal text is in [`LICENSE`](LICENSE).

You are free to:

- Use the software for any purpose  
- Study and modify the source code  
- Self-host and deploy  
- Contribute to the project  

**Condition:** If you run this software as part of a service (for example SaaS), you must make the **complete corresponding source code** publicly available, including any modifications.

This keeps improvements open and benefits the community.

---

## Commercial license

If you want to use this software **without** complying with AGPL obligations, you must obtain a **commercial license**.

This applies if you:

- Offer the software as a service without open-sourcing your code  
- Integrate it into a proprietary product  
- Embed it in a closed-source system  
- Want to avoid legal exposure from AGPL requirements  

**What you get:**

- Permission to use, modify, and distribute the software without open-source obligations under AGPL  
- Legal clarity for commercial use  
- Optional support or custom terms (if offered by the copyright holders)  

---

## Why this model exists

- Protect the project from being exploited by closed-source SaaS competitors  
- Encourage contributions and transparency  
- Enable fair commercial use through licensing  

If you are building an open ecosystem, **AGPL is your path**.  
If you are building a proprietary business, **contact us** for a commercial license.
